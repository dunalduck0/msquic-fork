# What Can Agentic Workflows Do?

Agentic workflows are AI-powered automations that help maintain and improve repositories. Here's what they can do in the **dunalduck0/msquic-fork** repository:

## Current Capabilities

### ‚úÖ **Daily Repository Status Reports**
The repository currently has one active agentic workflow:

- **File**: `.github/workflows/daily-repo-status.md`
- **What it does**: Creates daily issues summarizing repository activity
- **Includes**: Recent PRs, issues, discussions, releases, and code changes
- **Output**: GitHub issues with `[repo-status]` prefix
